# DSA Assignment

This repository contains implementations of various Data Structures and Algorithms in C.

## Programs

### Program 1: Balanced Parentheses Checker
Checks if mathematical expressions have balanced parentheses using stack.

**File:** `prog_1.c`  
**Documentation:** `doc_1.md`

### Program 2: Infix to Postfix Conversion and Evaluation
Converts infix expressions to postfix notation and evaluates them.

**File:** `prog_2.c`  
**Documentation:** `doc_2.md`

### Program 3: Reverse Traversal in Linked List
Implements reverse traversal of singly linked list using recursion.

**File:** `prog_3.c`  
**Documentation:** `doc_3.md`

### Program 4: Doubly Linked List
Implements doubly linked list with insert after node and delete operations.

**File:** `prog_4.c`  
**Documentation:** `doc_4.md`

### Program 5: Graph Traversal (BFS and DFS)
Implements graph using adjacency matrix and performs BFS and DFS traversal.

**File:** `prog_5.c`  
**Documentation:** `doc_5.md`

### Program 6: Min and Max Heap
Builds min heap and max heap from unsorted array.

**File:** `prog_6.c`  
**Documentation:** `doc_6.md`

### Program 7: Dijkstra's Shortest Path
Implements Dijkstra's algorithm to find shortest path in weighted graph.

**File:** `prog_7.c`  
**Documentation:** `doc_7.md`

### Program 8: Sorting Algorithms Comparison
Implements and compares multiple sorting algorithms with performance metrics.

**File:** `prog_8.c`  
**Documentation:** `doc_8.md`

## Compilation

To compile any program:
```
gcc prog_<number>.c -o prog_<number>
```

For programs using math library (prog_2):
```
gcc prog_2.c -o prog_2 -lm
```

## Running Programs

After compilation:
```
./prog_<number>
```

## Documentation

Each program has a corresponding documentation file (`doc_<number>.md`) that includes:
- Data structure definitions
- Function descriptions
- Main function organization
- Sample output
